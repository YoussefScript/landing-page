# NovaSync 🚀

**English** | [العربية](#novasync--1)

NovaSync is a modern, high-performance landing page for a fictional "Data Infrastructure for the AI Era" platform. It is fully responsive, supports seamless Dark/Light mode toggling, and includes internationalization (i18n) for both English and Arabic out of the box.

Designed and developed by **Youssef Emad Kamel**.

## 🌟 Features

- **Next.js App Router**: State-of-the-art React framework.
- **Tailwind CSS v4**: Utility-first styling with the latest configuration.
- **Dark & Light Mode**: Seamless theme switching powered by `next-themes`.
- **Internationalization (i18n)**: Full support for Left-to-Right (LTR) English and Right-to-Left (RTL) Arabic layouts.
- **Fully Responsive**: Carefully crafted layouts for Desktop, Tablet, and Mobile.
- **Icons**: Clean aesthetics using `lucide-react`.

## 🚀 Getting Started

First, install the dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

# NovaSync 🚀

**العربية** | [English](#novasync-)

NovaSync هي صفحة هبوط (Landing Page) حديثة وعالية الأداء لمنصة خيالية متخصصة في "البنية التحتية للبيانات في عصر الذكاء الاصطناعي". الصفحة متوافقة تماماً مع جميع الأجهزة (Responsive)، وتدعم التبديل السلس بين الوضع الليلي والنهاري، وتتضمن دعماً مدمجاً للغتين الإنجليزية والعربية (i18n).

تم التصميم والتطوير بواسطة **Youssef Emad Kamel**.

## 🌟 المميزات

- **Next.js App Router**: أحدث إطار عمل لـ React.
- **Tailwind CSS v4**: أحدث إصدار لتنسيق الواجهات بشكل سريع ومرن.
- **الوضع الليلي والنهاري**: تبديل سلس بين الألوان باستخدام `next-themes`.
- **دعم اللغات (i18n)**: دعم كامل للتخطيط من اليسار لليمين (الإنجليزية) ومن اليمين لليسار (العربية).
- **تصميم متجاوب بالكامل**: واجهات مصممة بعناية لتناسب أجهزة الكمبيوتر، الأجهزة اللوحية، والهواتف الذكية.
- **أيقونات عصرية**: استخدام مكتبة `lucide-react`.

## 🚀 طريقة التشغيل

أولاً، قم بتثبيت الاعتماديات (Dependencies):
```bash
npm install
```

ثم قم بتشغيل خادم التطوير:
```bash
npm run dev
```
افتح الرابط [http://localhost:3000](http://localhost:3000) في متصفحك لرؤية النتيجة.
